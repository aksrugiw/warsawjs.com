## How to build

You must pre-install `ruby` and `rubygems`.

```bash
$ rvm gemsets create www.warsawjs.com
$ rvm use 2.2.1@www.warsawjs.com
$ gem install bundler
$ bundle install
$ jekyll serve
```

## Launch

Open in browser: http://0.0.0.0:4000/
