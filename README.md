# warsawjs.com

> Use `GitHub Pages` to host homepage: https://warsawjs.com/

## Development

You must pre-install `rvm`.

<kbd>Copy + paster</kbd>

```bash
rvm gemsets create www.warsawjs.com
rvm use 2.2.1@www.warsawjs.com
gem install bundler
bundle install
jekyll serve
```
